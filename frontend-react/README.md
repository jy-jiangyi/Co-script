# README

## 1 Dependency and Environment
get into /frontend-react.

1. install node_modules: Run `npm install`
2. install react-router-dom@6 `npm install react-router-dom@6`
3. install ant-design: Run `npm install antd --save`
4. `npm rebuild`

## 2 update dependency
`npm rebuild`

## 3 Run frontend
`npm run dev`
